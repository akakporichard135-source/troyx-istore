# Admin Architecture

The admin dashboard route is ready for:

- Revenue, order, customer, and inventory metrics.
- Product create, edit, delete, image upload, bulk import, and export.
- Stock management and low-stock alerts.
- Coupons, promotions, homepage banners, reports, and audit logs.
- Store, shipping, email, tax, security, and role settings.

Protect admin routes with Supabase Auth claims or Firebase custom claims before enabling writes.
